# TaskFlow

TaskFlow is a modern, responsive, and beautifully designed task management application built with React, Vite, and Tailwind CSS.

## Features

- **Modern UI**: Built with responsive and premium design aesthetics using Tailwind CSS.
- **Fast and Lightweight**: Powered by Vite and React for a lightning-fast development and user experience.
- **Task Management**: Organize and manage your daily tasks seamlessly.

## Getting Started

### Prerequisites

- Node.js (version 18 or above recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) for icons
